# 🛒 E-Commerce Review Intelligence System | AI-Powered Amazon Sentiment Analytics Dashboard

An AI-powered analytics system that processes **Amazon product reviews** to extract actionable business insights using **Natural Language Processing (NLP), Machine Learning, and Data Visualization**.

The system analyzes customer feedback using both **VADER sentiment analysis (rule-based NLP)** and a **Machine Learning model (TF-IDF + Logistic Regression)**, and presents insights through an interactive **Streamlit dashboard**.

👉 **Live Demo:** https://e-comproreviewai.streamlit.app/

---

## 🚀 Key Features

- Real-time Sentiment Prediction (VADER + ML Comparison)
- Interactive Analytics Dashboard
- Sentiment Distribution Analysis
- Rating Distribution Insights
- Top Complaint & Praised Keywords
- Product Performance Analytics
- Category-Level Insights
- Review Search & Filtering
- Business Insights 

---

## 📊 Dataset

This project uses an **Amazon Product Reviews dataset**, which includes:

- Product names
- Categories
- Customer ratings
- Review text and titles
- Review timestamps
- Recommendation data

The dataset simulates real-world e-commerce feedback at scale.

---

## 🧠 Tech Stack

- Python 
- Pandas
- NLTK (VADER Sentiment Analysis)
- Scikit-learn (TF-IDF + Logistic Regression)
- Streamlit (Dashboard UI)
- Plotly (Data Visualization)

---

## ⚙️ System Architecture

1. Data Loading (Amazon Reviews Dataset)
2. Text Preprocessing (Cleaning, Stopword Removal, Lemmatization)
3. Sentiment Analysis (VADER + ML Model)
4. Feature Extraction (TF-IDF)
5. Dashboard Visualization (Streamlit + Plotly)
6. Business Insight Generation

---

## 📈 ML Model

- Algorithm: Logistic Regression
- Feature Extraction: TF-IDF Vectorizer
- Approach: Supervised Learning
- Output Classes:
  - Positive 😊
  - Negative 😡
  - Neutral 😐
